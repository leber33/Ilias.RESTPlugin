ILIAS Rest Plugin

An ILIAS Plugin for creating and using REST APIs

View project on GitHub

This is an extensible REST interface that allows developers to create RESTful APIs on top of the ILIAS LMS.

Features:

  • CRUD (Create-Read-Update-Delete) principle for resources
  • Permission management for resources depending on REST clients
  • Full OAuth 2.0 support (see RFC6749) including the grant types:
    • Authorization Code
    • Implicit
    • Resource Owner Password Credentials
    • Client Credentials
  • Easy integration of new REST endpoints
  • Based on the PHP SLIM Framework
